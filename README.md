# Maintenance of functional repeat arrays on the Drosophila Y chromosome

This repository contains supplementary files for the study "Maintenance of functional repeat arrays on the Drosophila Y chromosome" The scripts are organized into directories corresponding to the major steps of the methods section. Each main directory contains a README file with more specific details about its contents. 
## Repository Structure

Listed below are the main directories and the analyses they correspond to

### 1-Assembly and Validation
This directory contains all scripts related to the assembly and read validation of our three Y chromosome assemblies
### 2-Gene and Repeat Annotation
This directory contains all scripts related to the manual curation of the 13 major Y-linked fertility genes in our ISO-1 assembly, their validation with A4 Male and Female Nanopore direct RNA sequencing data, and the annotation of repetitive sequences in our three assemblies
